واجهة متجر إلكتروني متكاملة وعصرية لـ **شركة عمران التجارية** المتخصصة في تجارة وتوزيع لعب الأطفال والألعاب التعليمية. تم تصميم المتجر بأسلوب ناعم ومرح يعكس ثقة الأهل وجاذبية الأطفال، مع الاعتماد الكامل على لغات الويب الأساسية دون تعقيد.

---

## 🌟 المميزات الرئيسية (Key Features)

* **تصميم متجاوب بالكامل (Responsive Design):** يتكيف تلقائياً مع جميع الشاشات (الهواتف، الأجهزة اللوحية، والشاشات الكبيرة).
* **دعم لغوي كامل (RTL Support):** محاذاة وتصميم مخصص للغة العربية من اليمين إلى اليسار.
* **نظام متغيرات CSS حديث (`:root`):** سهولة تعديل الألوان العامة (البرتقالي المرح والكحلي الملكي) والظلال في مكان واحد.
* **تجربة مستخدم تفاعلية (Interactive UI):**
  * مؤثرات بصريّة أنيقة عند حركية الماوس (Hover Transitions).
  * شريط بحث ذكي وزر إضافة للمفضلة.
  * عداد كمية تفاعلي وجداول سلة تسوق منظمة.
* **صفحات متجر متكاملة:**
  * **الرئيسية (`index.html`):** البانر الترويجي، كروت التصنيفات، والمنتجات الأكثر مبيعاً.
  * **تفاصيل المنتج (`product-details.html`):** معارض الصور، التقييمات، والوصف.
  * **سلة التسوق (`cart.html`):** جدول المنتجات وملخص الطلب الحسابي.

## 🛠️ التقنيات المستخدمة (Tech Stack)

| التقنية | الاستخدام |
| :--- | :--- |
| **HTML5** | الهيكل البنائي للصفحات مع وسم الدلالات (Semantic Tags) |
| **CSS3 (Pure CSS)** | التنسيقات العامة، المتغيرات، والحركات التفاعلية |
| **Bootstrap 4.6** | تصميم الشبكة المرنة (Grid System) وعناصر الواجهة |
| **FontAwesome 5** | أيقونات واجهة المستخدم الاحترافية |
